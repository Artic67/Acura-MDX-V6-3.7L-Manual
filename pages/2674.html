Power Tilt/Telescopic Steering Control Unit Input Test

Power Tilt/Telescopic Steering Control Unit Input Test

NOTE: Before testing, troubleshoot the multiplex integrated control system first using B-CAN System Diagnosis Test Mode A.

1. Turn the ignition switch OFF.
2. Remove the steering column cover.




3. Disconnected the 16P connector (A) and 8P connector (B) from the power tilt/telescopic steering control unit (C).
4. Inspect the connector and socket terminals to be sure they are all making good contact.
- If the terminals are bent, loose or corroded, repair them as necessary and recheck the system.
- If the terminals look OK, go to step 5.




5. Reconnect the connectors to the power tilt/telescopic steering control unit, and make these input tests at the connectors.
- If any test indicates a problem, find and correct the cause, then recheck the system.
- If all the input tests prove OK, the control unit must be faulty; replace it.
