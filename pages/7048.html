DTC 16-3

DTC 16-3: Steering Angle Sensor Neutral Position Incorrect

NOTE:
^ Before you troubleshoot, review the general troubleshooting information.
^ Check the wheel alignment.
^ Check for installation of the steering angle sensor.

1. Turn the ignition switch ON (II).
2. Check for DTCs in the VSA system with the HDS.

Are any steering angle sensor DTCs indicated?

YES - Go to the indicated DTC's troubleshooting.

NO - Go to step 3.

3. Start the engine.
4. Check the STEERING ANGLE SENSOR in the DATA LIST with the HDS.
When the steering wheels is turned slowly, does the value change?

YES - Go to step 5.

NO - Update the SH-AWD control unit if it does not have the latest software or substitute a known-good SH-AWD control unit and recheck. If the symptom/indication goes away with the updated SH-AWD control, unit, troubleshooting is complete. If the symptom/indication goes away with a known-good SH-AWD control unit, replace the original SH-AWD control unit.

5. Test-drive the vehicle in a straight line at 19 mph (30 km/h) or more.
6. Park the vehicle on a level surface, and center the steering wheel.
7. Check the STEERING ANGLE SENSOR in the DATA LIST with the HDS.
It is Within ±25 deg of center?

YES - Intermittent failure, the system is OK at this time. If the DTC returns, replace the SH-AWD control unit.

NO - Do the steering angle sensor center point writing. If the DTC returns, replace the SH-AWD control unit.
