Vehicle position icon constantly leaves road, moves erratically, or is very far from actual position

Vehicle position icon constantly leaves road, moves erratically, or is very far from actual position

NOTE:
- Check for aftermarket accessories that could interfere with the GPS signal.
- Check that the GPS antenna is plugged in.

1. Check the GPS icon on the navigation picture.

Is the GPS icon white?

YES - Do the troubleshooting for GPS icon is white or not shown.

NO - Go to step 2.

2. Go into the Diagnostic Menu, and use the "Yaw Rate" test to check the yaw rate sensor.
3. Go into the Diagnostic Menu, and use the "Car Status" test to check the vehicle speed pulse.

Are the yaw rate sensor and vehicle speed pulse OK?

YES - The problem may be normal. Check to see if the problem occurs in the same place. If it does, the problem could be in the database. Go to step 4.

NO - If the problem is the yaw rate sensor, replace the navigation unit. If the problem is the vehicle speed pulse, check for an open in the wire between the navigation unit and the PCM. If the wire is OK, troubleshoot the vehicle speed signal circuit.

4. Substitute a known-good navigation unit, and check to see if the problem occurs in the same place.

Does the problem occur in the same place?

YES - The problem is in the database. Report the problem according to the Navigation System Manual under "Reporting Errors."

NO - Replace the original navigation unit.
