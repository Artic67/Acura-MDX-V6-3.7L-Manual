Instrument Fascia Removal/Installation

Instrument Fascia Removal/Installation

Special Tools Required
KTC trim tool set SOJATP2014 *
* Available through the American Honda Tool and Equipment Program

NOTE:
- Take care not to scratch the dashboard and its related parts.
- Use the appropriate tool from the KTC trim tool set to avoid damage when removing components.

1. Tilt the steering column down, and telescope it out.
2. Remove these items.
- Driver's outer dashboard trim
- Driver's inner dashboard trim




3. Remove the instrument fascia (A).
1. Gently pull out the bottom to release the clips
2. Disconnect the select/reset switch connector (B).

4. Install the instrument fascia in the reverse order of removal, and note these items:
- If the clips are damaged or stress-whitened, replace them with new ones.
- Make sure the connector is plugged in properly.
- Push the clips into place securely.
