PGM-FI System

DTC U0155: F-CAN Malfunction (Powertrain Control Module (PCM)-Gauge Control Module)




General Description
The controller area network (CAN) transmits/receives pulsing signals to/from the control modules simultaneously by using two signal lines (CANH and CANL).

When the powertrain control module (PCM) does not receive the signals from the gauge control module via the CAN lines for more than a set time, the PCM detects a malfunction and a DTC is stored.




Monitor Execution, Sequence, Duration, DTC Type, OBD Status




Enable Conditions

Malfunction Threshold
No signals from the gauge assembly via the CAN lines are received for at least 1 second.

Diagnosis Details

Conditions for illuminating the MIL
When a malfunction is detected, the MIL comes on and the DTC and the freeze frame data are stored in the PCM memory.

Conditions for clearing the MIL
The MIL will be cleared if the malfunction does not recur during three consecutive trips in which the diagnostic runs.

The MIL, the DTC, and the freeze frame data can be cleared by using the scan tool Clear command or by disconnecting the battery.
