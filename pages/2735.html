Navigation Cannot Control XM Radio

Navigation cannot control XM radio

Diagnostic Test: Navi System Link

1. Make sure the anti-theft code for the audio system is entered.
2. Go into the Diagnostic mode, and use the "Navi System Link" diagnostic.

Is the "XM" icon red?

YES - Do the troubleshooting for the voice guidance cannot be heard.

NO - Go to step 3.

3. Substitute a known-good navigation unit.

Can the navi control XM radio?

YES - Replace the navigation unit.

NO - Do the audio system troubleshooting.
