Immobilizer Indicator Does Not Go Off

Immobilizer indicator does not go off

1. Turn the ignition switch OFF.
2. Connect the HDS to the data link connector.
3. Turn the ignition switch ON (II).
4. Enter the IMMOBILIZER, then select the IMMOBILIZER INFORMATION.
5. Do the SYSTEM CHECK with the HDS.

Is N-1 OK indicated?

YES - Do the gauge control module self-test. If the gauge control module is faulty, replace it.

NO - Substitute a known-good immobilizer-keyless control unit, then register it and recheck. If the symptom goes away, replace the original immobilizer-keyless control unit.
