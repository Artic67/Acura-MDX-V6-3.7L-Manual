DTC LIst - Multiplex Integrated Control

Multiplex Integrated Control System

DTC Troubleshooting Index

NOTE: Record all DTCs, and sort them by DTC type using the DTC troubleshooting index, then troubleshoot the DTC(s) in this order:

- Battery voltage DTCs
- Internal error DTCs
- Loss of communication DTCs (beginning with the lowest number first; for example, if B1005 and B1016 are retrieved, troubleshoot B1005 first).
- Signal error DTCs
