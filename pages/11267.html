Dashboard Side Vent

Dashboard Side Vent Removal/Installation

Special Tools Required
KTC trim tool set SOJATP2014 *
* Available through the American Honda Tool and Equipment Program

NOTE:
- Take care not to scratch the dashboard and its related parts.
- Use the appropriate tool from the KTC trim tool set to avoid damage when removing components.
- The driver's side vent is shown; the passenger's side vent is similar.

1. Remove these items:
- A-pillar trim.
- Driver's side : Driver's dashboard outer trim.
- Passenger's side: Passenger's dashboard trim.




2. Remove the screws (A).
3. Pull out the side vent (B) to release the clip (C), then remove the vent.
4. Install the side vent in the reverse order of removal, and push the clip into place securely.
