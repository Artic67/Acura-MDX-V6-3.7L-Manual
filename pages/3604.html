Driver's Outer Dashboard Trim

Driver's Outer Dashboard Trim Removal/Installation

Special Tools Required
KTC trim tool set SOJATP2014 *
* Available through the American Honda Tool and Equipment Program

NOTE:
- Put on gloves to protect your hands.
- Use the appropriate tool from the KTC trim tool set to avoid damage when removing components.
- Take care not to scratch the dashboard and its related parts.




1. Open the front door, and remove the driver's dashboard side cover (A).
1. Pull out the front door opening seal (B), as needed.
2. Gently pull out the front edge of the side cover, and pry the rear edge with a trim tool to release the hooks (C), then remove the side cover.




2. Gently pull out the driver's outer dashboard trim (A) to detach the clips.
3. Disconnect the power mirror switch connector (B), VSA OFF switch connector (C), and if equipped, headlight washer switch connector (D), then remove the panel.
4. Install the trim in the reverse order of removal, and note these items:
- Make sure each connector is plugged in properly.
- If the clips are damaged or stress-whitened, replace them with new ones.
- Push the clips into place securely.
