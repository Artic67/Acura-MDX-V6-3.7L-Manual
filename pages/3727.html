Instrument Panel, Gauges and Warning Indicators: Description and Operation

Gauges and Indicators

How the Circuit Works

The power supply circuit/controller area network controller built into the gauge control module is part of the multiplex integrated control system. The B-CAN and F-CAN multiplex communication lines communicate information from specific system control units to the gauge control module power supply circuit/controller area network controller controlling the following indicators:

- ABS Indicator
- Active Damper System Indicator (USA: Sport Package, Sport/Entertainment Package; Canada: Elite Package)
- A/T Temp Indicator
- Charging System Indicator
- Fog Light Indicator
- High Beam Indicator
- Immobilizer Indicator
- Lights On Indicator
- Low Oil Pressure Indicator
- Low Tire Pressure Indicator
- MIL Indicator
- P, R, N, D, M, D3 Indicators
- Seat Belt Reminder Indicator
- Security Indicator
- SH-AWD Indicator
- Side SRS Off Indicator
- SRS Indicator
- TPMS Indicator
- VSA Activation Indicator
- VSA Indicator

Refer to each individual system for more information.

Engine Coolant Temperature Gauge

The power supply circuit/controller area network controller built into the gauge control module controls the engine coolant temperature gauge. The PCM provides engine coolant temperature information to the power supply circuit/controller area network controller through the F-CAN network. The PCM receives engine coolant temperature information from the two ECT sensors.

Fuel Gauge and Low Fuel Indicator

WARNING: Do not smoke while working on the fuel system. Keep open flame away from the work area. Drain fuel only into an approved container.

The power supply circuit/controller area network controller built into the gauge control module controls the fuel gauge and low fuel indicator. The power supply circuit/controller area network controller receives fuel level information from the fuel gauge sending unit. The resistance through the fuel gauge sending unit changes with the level of the fuel in the fuel tank. The power supply circuit/controller area network controller determines the fuel level from the varying resistance. The low fuel indicator illuminates when the fuel level is low. Turning the ignition switch to LOCK (0), returns the indicator needle to the bottom of the gauge dial regardless of the fuel level.

Speedometer

The power supply circuit/controller area network controller built into the gauge control module controls the speedometer. The PCM provides vehicle speed information to the power supply circuit/controller area network controller through the F-CAN network. The PCM receives vehicle speed information from the output shaft (countershaft) speed sensor.

Tachometer

The power supply circuit/controller area network controller built into the gauge control module controls the tachometer. The PCM provides engine speed information to the power supply circuit/controller area network controller through the F-CAN network. The PCM receives engine speed information from the CKP sensors (A and B).
