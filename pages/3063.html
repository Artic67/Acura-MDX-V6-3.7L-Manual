Center Console Panel

Center Console Panel Removal/Installation

Special Tools Required
KTC trim tool set SOJATP2014 *
* Available through the American Honda Tool and Equipment Program

NOTE:
- Take care not to scratch the console.
- Use the appropriate tool from the KTC trim tool set to avoid damage when removing components.

1. Remove the center console.
2. Remove the center console latch panel.




3. Pull the center console panel (A) out to detach the clips (B, C). Disconnect the accessory power socket connector (D) and seat heater switch connectors (E), then remove the panel.




4. If necessary, remove the screws, then remove the console pocket (A) from the console panel (B).
5. Install the panel in the reverse order of removal, and note these items:
- Make sure each connector is plugged in properly.
- If the clips are damaged or stress-whitened, replace them with new ones.
- Push the clips into place securely.
